This project is a frontend web application for an online food ordering system called Master or Rahul Curry Point. It is developed using HTML, CSS, and JavaScript, and it simulates how a real restaurant website works. In this project, users can browse food items, register an account, log in, add items to a cart, and place orders. The project mainly has three parts: the home page, the login and registration system, and the cart and ordering system.
The home page is designed to look attractive and user-friendly. It includes a navigation bar with options like Home, Menu, About, and Contact, along with an Order Now button. The hero section displays a tagline “HOT SPICY HOME-STYLE” and highlights the message “Taste the Tradition.” It also shows popular dishes like Butter Chicken, Dal Makhani, Biryani, and Paneer Tikka with their prices. There is also an About section that explains the restaurant’s story, experience, and features like authentic spices, fresh ingredients, and family recipes. At the bottom, the footer provides contact details, working hours, and useful links.
The login and registration system allows users to create an account and log in. During registration, the user enters details like name, email, and password, and the system checks for valid input such as correct email format and matching passwords. The data is stored in localStorage. During login, the system checks the entered details with the stored data and allows access if the details are correct.
The cart and ordering system is the main functionality of the project. Users can add items to the cart, increase or decrease quantity, and remove items. The cart data is stored in localStorage so it remains even after refreshing the page. A sidebar shows selected items, their quantity, price, and total amount. When the user places an order, a random order ID is generated, and the cart is cleared.
Overall, this project is simple, easy to understand, and shows how a real food ordering system works using only frontend technologies.
